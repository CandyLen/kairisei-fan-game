# 괴리성 밀리언아서 - 덱빌딩 & PVE 프로토타입

괴리성 밀리언아서의 게임 시스템을 참고한 웹 기반 덱빌딩 + PVE 전투 프로토타입입니다.

## 주요 시스템
- **4직업**: 용병(물리딜러) / 부호(탱커·드로우) / 도적(마법딜러) / 가희(힐러·버퍼)
- **76장+ 카드 DB**: 원작 카드명 기반, MR/UR/SR 등급
- **가챠 시스템**: MR 3% / UR 15% / SR 82%, 1회·11연차 뽑기 + 연출 애니메이션
- **덱 빌딩**: 10장 덱 구성, 리더 카드 HP×1.5 보너스
- **턴제 PVE 전투**: 코스트 시스템, 4체인, AI 파티, 보스 레이드 (부위 파괴)
- **드로우 시스템**: 부호의 드로우 카드로 패 순환

## 조작법
1. 퀘스트 → 직업 선택
2. 덱 편성 → 카드 10장 선택 → 출격
3. 전투 → 보스 선택 → 카드 제출 → 실행
4. 뽑기 → 새 카드 획득

## 기술 스택
- 단일 HTML 파일 (외부 의존성: Google Fonts만 사용)
- Vanilla JS, CSS Grid/Flexbox
- 모바일 가로(4:3) 최적화
